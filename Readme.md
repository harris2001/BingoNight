# This is a simple web-based app for bingo events

## Instructions:
Building API:
```bash
node install express
node .\index.js
```
Running the service
```bash
python3 -m http.server 8080 --directory .
```
Live update on /new_round page
```bash
#Manually change the contents of balls.json
e.g. ["3","45","12"]
```
